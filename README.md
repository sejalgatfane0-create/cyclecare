# CycleCare 🌸

CycleCare is a web-based health monitoring system designed to help users track and manage their menstrual cycles efficiently.

## 📌 Features
- Track menstrual cycles
- Predict upcoming periods
- Monitor health patterns
- Simple and user-friendly interface

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

## 🚀 How to Use
1. Open the application in your browser
2. Enter your cycle details
3. View predictions and health insights

## 🔮 Future Scope
- AI-based cycle prediction
- Mobile application development
- User login & data storage
- Notifications and reminders

## 📷 Project Structure
- index.html → Main webpage  
- style.css → Styling  
- script.js → Functionality
